Sitio web oficial del gobierno: Los sitios web oficiales usan .gov.

Un sitio web .gov pertenece a una organización gubernamental oficial de los Estados Unidos.

Sitio web seguro: Los sitios web .gov seguros usan HTTPS.

Un candado ( ) o https:// significa que te has conectado de forma segura al sitio web .gov. Comparte información confidencial solo en sitios web oficiales y seguros.
